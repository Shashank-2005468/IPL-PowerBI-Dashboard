# IPL Analysis Dashboard (2008–2025)

## Project Overview
This project analyzes IPL data using Power BI.

## Features
- Total Matches
- Total Runs
- Total Fours & Sixes
- Total Wickets
- Team-wise Wins
- Venue Analysis
- Champion & Runner-Up
- Orange Cap & Purple Cap Winners
- Player Performance Analysis

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV Dataset
